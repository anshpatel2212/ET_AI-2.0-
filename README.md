# ­¤ōĪ Data Accuracy Strategy

This platform is designed to avoid fake or untraceable air-quality values.

## Source Priority

### Air Quality
1. **CPCB official data** for India
2. **Google Air Quality API**
3. **WAQI**
4. **OpenAQ**
5. **IoT sensor layer** (if deployed)

### Weather
1. OpenWeather
2. IMD backup

## Validation Rules
- Compare multiple sources
- check freshness of timestamps
- verify station/coordinate match
- detect stale cache
- calculate confidence score
- flag mismatch when AQI delta > 20

## Confidence Score Logic
- **95ŌĆō100%** = official + strong cross-match
- **85ŌĆō94%** = trusted agreement
- **70ŌĆō84%** = partial agreement
- **Below 70%** = low confidence warning

Each major AQI widget should show:
- source used
- last updated time
- confidence percentage
