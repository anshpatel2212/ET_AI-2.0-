## 🌍 Overview

**AI-Powered Urban Air Quality Intelligence for Smart City Intervention** is a full-stack intelligent platform designed to monitor, analyze, predict, and respond to urban air pollution in real time.

It combines:

- **Government AQI data**
- **Google Air Quality validation**
- **IoT sensor feeds**
- **Weather data**
- **Traffic overlays**
- **Satellite signals**
- **AI forecasting**
- **Smart intervention recommendations**

The system is built for:

- **Citizens**
- **Health-sensitive users**
- **Municipal officers**
- **Pollution control boards**
- **Smart city control rooms**
- **Urban researchers**

# 📡 Data Accuracy Strategy

This platform is designed to avoid fake or untraceable air-quality values.

## Source Priority

### Air Quality
1. **CPCB official data** for India
2. **Google Air Quality API**
3. **WAQI**
4. **OpenAQ**
5. **IoT sensor layer** (if deployed)

### Weather
1. OpenWeather
2. IMD backup

## Validation Rules
- Compare multiple sources
- check freshness of timestamps
- verify station/coordinate match
- detect stale cache
- calculate confidence score
- flag mismatch when AQI delta > 20

## Confidence Score Logic
- **95–100%** = official + strong cross-match
- **85–94%** = trusted agreement
- **70–84%** = partial agreement
- **Below 70%** = low confidence warning

Each major AQI widget should show:
- source used
- last updated time
- confidence percentage
